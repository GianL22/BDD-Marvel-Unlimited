# Marvel Unlimited API GRAPHQL

Pasos para ejecutar el backend en un entorno de desarrollo por primera vez:

1. Crear un archivo .env en el root con las siguientes variables (preguntar al due√±o del repositorio por los valores de las variables)
```
DB_PASSWORD=
DB_NAME=
DB_HOST=
DB_PORT=
DB_USERNAME=
PORT= 3000
JWT_SECRET=
```

2. Ejecutar el servidor (En el puerto 3000)
```
npm run start:dev
```
