# Ultimate Marvel
1. Descargar las dependencias del backend
```sh
cd backend
npm i 
```
2. Descargar las dependencias del frontend
```sh
cd frontend
yarn
```
3. Crear .env en backend
```sh
cd backend
PORT = 
```
4. Ejecutar frontend
```sh
cd frontend
yarn dev
```
5. Ejecutar backend
```sh
cd backend
npm run start:dev
```
